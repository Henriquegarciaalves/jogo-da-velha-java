# 🎮 Jogo da Velha em Java

Projeto simples de Jogo da Velha feito em Java por **Henrique Alves** para praticar lógica de programação.

---

## 📌 Sobre o projeto

O jogo funciona em um tabuleiro 3x3 usando matriz.  
Dois jogadores alternam entre **X e O**, e o programa verifica automaticamente se alguém venceu.

---

## 🧠 Regras

- Jogadores jogam alternadamente (X e O)
- Não pode escolher posição ocupada
- Não pode sair do tabuleiro
- O jogo termina quando alguém vence
